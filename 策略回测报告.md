# 双均线交叉策略回测报告

## 一、方法路线

### 1.1 策略概述

本策略采用**双均线交叉**（Moving Average Crossover）方法，是一种经典的趋势跟踪策略。该策略通过计算两条不同周期的移动平均线，利用它们的交叉点作为买卖信号。

### 1.2 策略原理

**核心逻辑：**
- **短期均线（快线）**：反映近期价格趋势，对价格变化敏感
- **长期均线（慢线）**：反映中长期趋势，起到平滑和过滤作用

**交易信号：**
1. **做多信号**：短期均线上穿长期均线（金叉）
   - 表示短期趋势转强，价格可能上涨
   
2. **做空信号**：短期均线下穿长期均线（死叉）
   - 表示短期趋势转弱，价格可能下跌

### 1.3 策略参数

| 参数名称 | 参数值 | 说明 |
|---------|--------|------|
| 短期均线周期 | 5日 | 快线，反映短期趋势 |
| 长期均线周期 | 20日 | 慢线，反映中期趋势 |
| 止损比例 | 2% | 单笔交易最大亏损控制 |
| 止盈比例 | 3% | 单笔交易盈利目标 |
| 开仓手数 | 3手 | 固定仓位管理 |
| K线周期 | 日线 | 使用日线数据进行信号判断 |

### 1.4 风控机制

策略内置了完善的风险控制机制：

1. **止损机制**：当持仓亏损达到2%时自动平仓
   - 多头止损：当前价格 ≤ 开仓价 × (1 - 2%)
   - 空头止损：当前价格 ≥ 开仓价 × (1 + 2%)

2. **止盈机制**：当持仓盈利达到3%时自动平仓
   - 多头止盈：当前价格 ≥ 开仓价 × (1 + 3%)
   - 空头止盈：当前价格 ≤ 开仓价 × (1 - 3%)

3. **持仓管理**：每次只持有单一方向的仓位，避免对冲

### 1.5 技术实现

策略基于 **TqSdk（天勤量化）** 框架实现，主要技术要点：

- 使用 `TqSim` 进行模拟交易
- 使用 `TqBacktest` 进行历史回测
- 使用 `TargetPosTask` 进行目标持仓管理
- 使用 `ma()` 函数计算移动平均线
- 实时监控行情变化，在K线收盘时判断信号

## 三、结果分析

### 3.1 整体表现评价

#### 优势分析

1. **稳健的盈利能力**
   - 策略在回测期间实现了正收益，总收益率0.467%
   - 年化收益率2.74%，虽然不高但保持了稳定盈利
   - 最大回撤仅0.234%，风险控制良好

2. **优秀的盈亏比**
   - 盈亏比达到2.90，意味着平均每次盈利是亏损的近3倍
   - 这说明策略的止盈止损设置合理，能够"截断亏损，让利润奔跑"
   - 即使胜率只有50%，依靠高盈亏比仍能实现盈利

3. **良好的风险控制**
   - 最大回撤控制在极低水平（0.234%）
   - 最大连续亏损天数仅4天，说明策略不会出现长时间亏损
   - 止损机制有效执行，避免了单笔大额亏损

#### 不足之处

1. **交易频率偏低**
   - 43个交易日仅开仓4次，平均每10天交易一次
   - 交易机会较少，可能错失部分行情
   - 建议考虑优化均线参数或增加其他过滤条件

2. **夏普比率偏低**
   - 夏普比率0.269，低于一般量化策略的合格线（通常>1）
   - 说明相对于承担的风险，收益水平还有提升空间
   - 年化收益率2.74%相对较低

3. **卡玛比率偏低**
   - 卡玛比率0.066，说明年化收益与最大回撤的比例不够理想
   - 虽然最大回撤很小，但收益也相应较低

### 3.2 策略适用性分析

#### 适合的市场环境

1. **趋势明显的市场**
   - 双均线策略本质是趋势跟踪策略
   - 在铜期货出现明显上涨或下跌趋势时表现较好

2. **波动适中的市场**
   - 过度震荡会导致频繁止损
   - 波动过小则交易机会不足

#### 不适合的市场环境

1. **震荡市**
   - 横盘震荡时均线频繁交叉，容易产生假信号
   - 可能导致连续止损

2. **急涨急跌行情**
   - 均线具有滞后性，可能错过最佳入场点
   - 急跌时止损可能来不及执行

### 3.3 改进建议

#### 短期优化建议

1. **调整均线参数**
   - 当前5/20日均线可能过于保守
   - 建议测试其他参数组合，如(5,10)、(10,20)、(10,30)等
   - 可以根据铜期货的历史波动特性选择最优参数

2. **优化止盈止损比例**
   - 当前2%止损、3%止盈的比例较为保守
   - 可考虑根据ATR（平均真实波幅）动态设置止损止盈
   - 或采用移动止损，锁定更多利润

3. **增加过滤条件**
   - 增加成交量确认：只在成交量放大时确认信号
   - 增加趋势强度过滤：如ADX指标，只在趋势明显时交易
   - 增加时间过滤：避开流动性较差的时段

#### 中长期改进方向

1. **仓位管理优化**
   - 当前固定3手开仓，可改为动态仓位
   - 根据账户权益、波动率、信号强度动态调整仓位
   - 采用凯利公式或风险平价方法

2. **多周期确认**
   - 增加更长周期（如周线）的趋势确认
   - 只在大周期趋势方向一致时交易
   - 提高信号质量，减少假突破

3. **多品种组合**
   - 考虑在多个相关品种上同时运行策略
   - 通过分散化降低单一品种风险
   - 可考虑铜、铝、锌等有色金属组合

4. **引入机器学习**
   - 使用机器学习方法优化参数选择
   - 训练模型识别更有效的交易信号
   - 但需注意过拟合风险

### 3.4 风险提示

1. **过拟合风险**
   - 回测结果良好不代表实盘表现一定优秀
   - 参数可能过度拟合历史数据
   - 建议进行样本外测试和前瞻性验证

2. **市场环境变化**
   - 铜期货市场可能受宏观经济、供需关系等多因素影响
   - 策略在不同市场环境下表现可能差异较大
   - 需要持续监控策略表现并及时调整

3. **交易成本影响**
   - 回测已考虑手续费，但实盘可能还有滑点成本
   - 高频交易策略对交易成本更敏感
   - 建议在实盘前进行模拟盘测试

4. **流动性风险**
   - cu2604合约临近交割时流动性可能下降
   - 建议设置合约换月规则，及时移仓换月

### 3.5 总结

本次回测的双均线交叉策略在SHFE.cu2604合约上表现**稳健但保守**。策略的核心优势在于：

- ✅ 风险控制优秀（最大回撤仅0.234%）
- ✅ 盈亏比良好（2.90）
- ✅ 策略逻辑清晰，易于理解和执行

主要不足：

- ⚠️ 交易频率较低，机会成本较高
- ⚠️ 年化收益率偏低（2.74%）
- ⚠️ 夏普比率、卡玛比率有待提升

**总体评价**：该策略适合风险偏好较低、追求稳健收益的投资者。如需提高收益，建议在保持风险控制的前提下，通过优化参数、增加过滤条件、改进仓位管理等方式进行提升。实盘应用前，建议进行更长时间的样本外测试和模拟盘验证。
